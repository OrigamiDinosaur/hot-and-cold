LeanTween is an efficient tweening engine for Unity3d.
This is a cut-down version of the Unity package from the Asset Store, based on release 2.51 from December 2021.
Editor scripts, Documentation and Examples have been removed, and an Assembly Definition file has been added.

Asset Store Package: https://assetstore.unity.com/packages/tools/animation/leantween-3595
Full Documentation:	http://dentedpixel.com/LeanTweenDocumentation/classes/LeanTween.html
