# Hot & Cold	

## Overview 

A simple treasure hunting game inspired by Final Fantasy IX's Chocobo Hot & Cold. 
Made primarly to gain some experience in the Unity Addressables sytem. 

In hindsight I should've just made a custom character creator or something similar, but its easier to develop when you're making something you enjoy. 

The game level and the equippable hats are addressables loading via Unity's CCD system. 

## Potential Updates

FFIX's Chocobo Hot & Cold is one my favourite minigames in gaming history so I would like to continue developing this if time allows.

Should I choose to do so these would be my focus points

- Custom art assets (currently using free online assets) 
- More feedback in the gameplay (player animations, vfx)
- More comprehensive upgrade paths
- Fully customizable player
- More levels (unlocking more currencies along the way)
- Make the currency system more modular. Started off with modularity in mind but when I settled on only two currencies for the demo I settled on a more straight forward approach.
